使用两个队列实现一个栈

思路：用两个队列实现一个栈，在入栈操作时只将元素插入到非空队列中，空的队列用来做出栈操作，出栈时要得到的元素是非空队列的最后一个元素，将非空队列（size为n）的前n-1个元素转移到空队列中，第n个元素即为出栈元素。
